### 一. 概述
+ Vue是一个构建用户界面的前端框架
+ Vue是一个轻量级的MVVM(Model-View—ViewModel)框架
+ 与angular、react类似，实现数据的双向绑定
+ 数据驱动 + 组件化的前端开发技术
+ 作者尤雨溪

### 二. 起步
#### 1. 下载核心库vue.js
+ CDN引入js `<script src="https://unpkg.com/vue@next"></script>`
+ 在目录下使用npm安装 
```
npm init --yes
cnpm install vue --save
```
